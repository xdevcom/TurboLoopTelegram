# 🚀 TurboLoop Telegram Marketing Bot v4.0 (180 Posts)

An ultra-premium, zero-dependency, automated Telegram marketing bot designed to post **180 unique premium marketing images and 180 unique English captions** for TurboLoop.

Every single post is automatically **sent and pinned** in your channel, ensuring maximum visibility.

---

## 📦 Package Contents

- `bot.py` - Main execution script containing all 180 unique captions hardcoded.
- `config.json` - Configuration file pre-filled with your credentials.
- `requirements.txt` - Required Python dependencies (only `requests`).
- `images/` - Directory containing **180 unique premium AI-generated images** with consistent TurboLoop branding and logo.

---

## ⚙️ How It Works

1. The bot cycles through **18 unique topics** in order.
2. For each topic, it selects one of **10 unique variations** (totaling 180 unique posts).
3. It picks the corresponding unique image and formatted English caption.
4. It posts the image + caption to your Telegram Channel.
5. It automatically **pins** the post in the channel.
6. It sleeps for **4 hours** (6 posts per day, total of 30 days of unique content).
7. Once all 180 posts are sent, it loops back to post #1 automatically.

---

## ⚡️ Quick Start

### 1. Install Dependencies
```bash
pip install -r requirements.txt
```

### 2. Run the Bot
```bash
python bot.py
```

### 3. Run 24/7 on Server (Background)
```bash
nohup python bot.py > logs/bot.log 2>&1 &
```
